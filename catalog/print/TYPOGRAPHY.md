## Headings

```type
{
	"color": "#000000",
	"headings": [
    { "label": "Heading 1", "value": 62 }
  ],
  "font": "SourceSansPro"
}
```

```type
{
	"color": "#000000",
	"headings": [
		{ "label": "Heading 2", "value": 42 },
		{ "label": "Heading 3", "value": 30 },

	],
  "font": "SourceSansPro"

```
